## Começando

Após baixar o projeto, digite os seguintes comandos no terminal

```bash
npm run dev
# or
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) com seu navegador para ver o resultado.

## Tecnologias usadas

- Next JS
- React JS
- Styled Components
- ESLint
- Prettier
